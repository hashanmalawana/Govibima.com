== Description ==

Discover RW is a modern responsive blog theme. This theme is suitable to use for blog, personal site, travel, informative, life style site and much more... It comes with many options as changing color scheme of the site, blog layouts, widget areas, , social links and more.

Responsive design

Blog post layout changing

Sociable Icons section in footer and Social widget

SEO Ready

Comments with reply functionality (multiple levels depth)

Works and looks similar in all major browsers: Internet Explorer, Firefox, Opera, Safari, Google Chrome

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
